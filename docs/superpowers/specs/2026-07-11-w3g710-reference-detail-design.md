# W3G710-NU31-03 参考页忠实版详情设计

## 目标

将 `W3G710-NU31-03` 当前简化 GEO 详情替换为用户提供参考页的忠实版结构，并把该结构固化到持久化上传器，使后续型号能够通过真实 PDF 参数、图纸和实拍图生成相同质量的详情页。

页面只使用当前型号资料，不使用其他型号图片，不编造客户案例、性能数据或使用效果。更新本地 HTML 和 1688 表单后，流程必须停在“保存草稿”前，不自动保存或发布。

## 视觉方向

采用已确认的方案 A“参考页忠实版”：

- 深蓝色 `#0b3a70` 作为标题和收尾强调色。
- 浅灰色 `#f6f8fb` 与白色内容区交替，形成长页面阅读节奏。
- 表格使用浅灰边框、蓝色表头或浅蓝字段列。
- 所有样式使用内联 CSS，兼容 1688 TinyMCE 编辑器。
- 页面保持单栏、全宽图片和清晰的 H1/H2/H3 层级。
- 不使用渐变、动画、脚本或依赖站外样式表。

## 页面结构与素材映射

页面按以下顺序生成：

1. **产品尺寸图**：从当前型号 PDF 第 4 页裁出真实产品图纸，上传到 1688 图片空间后使用其托管 URL。
2. **场景使用**：使用当前产品实拍图，配合设备散热、冷却系统、厂房通风、工业换气四项保守说明；不生成虚构场景照片。
3. **产品组成**：用当前型号整机、防护网、EC 电机、铭牌实拍图说明产品结构。
4. **型号标题与产品定义**：明确品牌、完整型号、电气类型和产品类别。
5. **买家为什么选择这款风机**：覆盖大风量、完整型号匹配、EC 调速控制、工业防护。
6. **核心参数**：三列表格展示项目、参数和选型含义。
7. **适用场景**：设备散热、冷却系统、厂房通风、工业换气。
8. **选型提醒**：电压、工作点、安装方向和安全防护。
9. **采购前请确认**：完整型号、电源、尺寸重量、数量与物流。
10. **常见问题 FAQ 与一句话选型**：覆盖型号、电气参数、风量静压、尺寸重量和安装确认。

详情至少包含五张当前型号真实图片：一张尺寸图和四张实拍图。图片 `alt` 必须包含品牌、完整型号和图片所示部位或用途。

## 上传器架构改造

### 详情模板

将当前简化的 `render_geo_detail` 改为参考页忠实模板。渲染器接收：

- 品牌、完整型号和标题；
- 结构化规格参数；
- 四张实拍图的 1688 托管 URL 与角色；
- 一张尺寸图的 1688 托管 URL；
- 适用场景、选型提醒、采购确认和 FAQ 数据。

渲染结果写入当前型号的 `automation/<MODEL>/detail.html`，随后注入 1688 TinyMCE，并调用页面模型同步逻辑。

### 详情素材

准备阶段从 PDF 指定页渲染并裁剪产品图纸，保留尺寸线、尺寸值和完整型号，去除无关页眉、页脚和大面积空白。生成文件放在当前型号资料目录的上传优化子目录中，原 PDF 和原图不覆盖。

上传阶段通过详情编辑器的图片选择器把尺寸图上传到当前配置相册，并取得 1688 托管 URL。尺寸图不能占用四张主图位置。四张实拍图继续复用主图上传后返回的托管 URL。

### 状态与失败边界

在任务状态中记录详情模板版本、尺寸图本地路径、尺寸图托管 URL、详情图片数量和最终 HTML 路径。

出现以下任一情况时停止详情更新并写入失败状态：

- PDF、产品图纸页或四张当前型号实拍图缺失；
- 型号不完全匹配；
- 核心电气参数、外形尺寸或重量缺失；
- 尺寸图无法上传或没有取得托管 URL；
- 最终 HTML 含其他型号、图片不足五张或出现空 `src`；
- TinyMCE 或页面模型同步失败。

失败时不得覆盖当前编辑器中的有效详情，不得点击保存或发布按钮。
